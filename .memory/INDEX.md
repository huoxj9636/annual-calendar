# 跨对话记忆索引

> **使用规则**：
> 1. 每次新对话开始，必须先读本文件扫描历史
> 2. 根据索引条目的"详情定位"列，用 `read_file(offset=X, limit=Y)` 精准跳读，不要通读全文
> 3. 每次对话结束时：① 创建详情文件（带目录+行号） → ② 追加索引条目（含精准定位） → ③ git push
>
> **索引生成规则**：索引条目必须从详情文件中提取，禁止凭 Git 历史猜测创建。
> **定位格式**：`文件名#章节名(L行号)`，例：`sessions/2025-01-14-周复盘时间轴.md#最终决策(L20)`
