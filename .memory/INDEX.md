# 跨对话记忆索引

> **使用规则**：
> 1. 每次新对话开始，必须先读本文件扫描历史
> 2. 需要了解某次对话的决策细节时，根据"详情文件"列读取 `.memory/sessions/{filename}`
> 3. 每次对话结束时，必须：① 创建详情文件 → ② 追加索引条目到本文件 → ③ git push
>
> **索引生成规则**：索引条目必须从详情文件中提取，禁止凭 Git 历史猜测创建。
