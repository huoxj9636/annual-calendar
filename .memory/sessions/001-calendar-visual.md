# Session 001 - 年历视觉打磨

## 日期
2025-01

## 需求
年历网格的视觉优化：格子布局、周末色彩、12周边框、过去日期显示。

## 放弃的方案
- 12周区块波浪线边框（反复调整多次后最终放弃，改为星星标记后又移除）
- 周末高饱和度色彩 → 降为莫兰迪风格

## 最终决策
- 格子布局：日期加大、农历缩小、增加留白
- 周末：当月主题色背景，色调偏莫兰迪
- 过去日期：背景灰色 + 文字淡化
- 标语"永远不要放弃"居中置于头部
