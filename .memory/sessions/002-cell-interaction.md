# Session 002 - 格子交互升级

## 日期
2025-01

## 需求
年历格子支持多种点击交互：勾选、备忘、日视图。

## 放弃的方案
- 三区域点击（左日视图/右打勾/下备忘）→ 太复杂
- 右上角小勾/叉标记 → 视觉杂乱

## 最终决策
- 格子改为上下两区点击：上半打开日视图、下半切换勾选
- 日视图侧边栏：24h时间线 + 语音识别 + 备忘
- 语音识别改为持续录音模式，手动点击停止
- 有日程/备忘的格子显示绿色笔记图标
- 勾选标记：半透明水印风格居中显示

## 涉及文件
- year-calendar.tsx, day-view.tsx
