# Session 009 - 知识森林

## 日期
2025-01

## 需求
类似蚂蚁森林的知识管理可视化，5种知识类型映射为不同树种。

## 放弃的方案
- 卡片森林风格（用UI元素堆叠模拟森林）→ DESIGN.md明确禁止
- emoji大树 → 禁止

## 最终决策
- 森林全景画布（4层透视：晨空/远山/中景/草地）
- 5种知识类型映射树种：橡树(root)/杉木(trunk)/枫(branch)/嫩枝(leaf)/樱花(fruit)
- 树摇曳/飘落叶/光斑动效
- 点击树进入单棵树特写
