## A1（7/12）— bug根因+修复

### 根因定位

在 `src/components/year-calendar.tsx` 第 349-358 行（修复前），加载 `calendar-module-order` 的逻辑：

```ts
const savedOrder = localStorage.getItem('calendar-module-order');
if (savedOrder) {
  try {
    const parsed = JSON.parse(savedOrder) as string[];
    const allKeys = Object.keys(defaultModuleNames);  // 只有8个内建ID
    const validOrder = parsed.filter(k => allKeys.includes(k));  // ← bm_xxx 被过滤掉
    const missing = allKeys.filter(k => !validOrder.includes(k));
    setModuleOrder([...validOrder, ...missing]);
  } catch { /* ignore */ }
}
```

**Bug 链路**：
1. 用户添加新链接 → `setModuleOrder(prev => [...prev, newId])` 把 `bm_xxx` 写进 localStorage ✓
2. `bookmarks` 状态从 localStorage 同步加载（line 240-242）✓
3. 但 `moduleOrder` 加载时用 `allKeys.includes(k)` 过滤，**只接受 8 个内建 ID**
4. `bm_xxx` 被当作"非法值"过滤掉
5. 侧边栏渲染（line 1109）`moduleOrder.filter(...)` 中没有 `bm_xxx`，所以不显示

### 修复方案

`year-calendar.tsx` 第 349-364 行，把 `validKeySet` 从「仅内建 8 个」扩展为「内建 8 个 + 所有书签 ID」：

```ts
const builtinKeys = Object.keys(defaultModuleNames);
const bookmarkIds = (() => {
  try { return (JSON.parse(localStorage.getItem('calendar-bookmarks') || '[]') as Array<{ id: string }>).map(b => b.id); }
  catch { return []; }
})();
const validKeySet = new Set([...builtinKeys, ...bookmarkIds]);
const validOrder = parsed.filter(k => validKeySet.has(k));
const missing = [...builtinKeys, ...bookmarkIds].filter(k => !validOrder.includes(k));
setModuleOrder([...validOrder, ...missing]);
```

### 验证

- `pnpm lint` ✓
- `pnpm ts-check` ✓
- 服务探活 5000 端口 ready ✓
- 手动走查：添加链接 → 写入 localStorage 三处（bookmarks/order/visibility）→ 刷新 → 全部正确加载

### 用户需求确认

> "我只有点击隐藏，它才会不显示"

修复后行为：
- 添加后立即显示 ✓
- 刷新后仍然显示 ✓
- 点击眼睛图标隐藏后，刷新也保持隐藏 ✓
- 删除书签后，刷新不显示 ✓

## A4（7/12）— 尺子效果实现

### 理解需求

用户要"尺子效果"：
- ❌ 不是数字（小时/15/30/45 数字标签）
- ✅ 是数字旁边的刻度竖线
- ✅ 底部要有一条连续横杠作为基线
- ✅ 竖线从横杠凸出来（像尺子）
- ✅ 竖线要更明显

### 现状问题

`daily-review.tsx` L800-830 头部刻度：
- 每个格子用 `border-r`（右边框）作为竖线
- 颜色用 `skin.cellBorder` = `rgba(148,163,184,0.10)`，**只有 10% 透明度，太淡**
- 没有横杠基线

### 修改

**头部刻度（L800-850）**：
1. 外层 flex 容器加 `border-b` + `borderBottomWidth: 2px` + `borderColor: skin.textMuted + '80'` → 连续横杠基线
2. 移除原来每个格子的 `border-r`，改为格子内 `absolute right-0 bottom-0` 的小竖线
3. 小时刻度线：宽 2px，高 10px，opacity 0.9 → 更高更粗
4. 分钟刻度线：宽 1px，高 6px，opacity 0.5 → 稍矮稍细
5. 颜色统一用 `skin.textMuted`（~90% 不透明灰）→ 明显可见

**甘特行内背景刻度线（L1553-1572）**：
- `borderColor: skin.cellBorder` → `skin.textMuted`
- `opacity-20` → `0.35`
- `borderRightWidth: 1.5px` → `1px`
- 与头部刻度视觉一致

### 验证

- `pnpm lint` ✓ / `pnpm ts-check` ✓
